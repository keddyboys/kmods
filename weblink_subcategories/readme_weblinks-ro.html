Subcategorii pentru legături sait v1.21 Modificare pentru PHP-Fusion v7.02.07
Ce face:

Această modificare permite crearea subcategoriilor pentru legături sait. Poţi creea subcategorii ale categoriilor curente. O categorie de legături sait poate avea atât subcategorie cât şi legăturăsait.
Instalare:
  1. Încarcă directorul install în directorul părinte.

  2. Intră pe http://www.siteultău.ro/install/index.php.

  3. După instalarea cu succes şterge directorul install.

    4. Trebuie să faci asta înainte de încarcarea sau modificarea fişierelor pentru prevenirea erorilor!
  4. Încarcă fişierele urmănd structura directoarelor.


Fişiere modificate:

    1. - weblinks.php,
      - administration/weblink_cats.php,
      - administration/weblinks.php,
      - locale/English/weblinks.php
      - locale/English/admin/weblinks.php,
  2. Coloana weblink_cat_parentad ăugată tablelului DB_WEBLINK_CATS.
  3. Adăugat fişierul includes/subcats_weblinks_include.php.
Dezinstalare:
  1. Încarcă directorul install în directorul părinte.

  2. Intră pe http://www.siteultău.ro/install/index.php

  3. După dezinstalarea cu succes şterge directorul install

  4. Poţi şterge fişierul includes/subcats_weblinks_include.php.

  5. Pune fişierele originale înapoi.
